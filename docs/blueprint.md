# **App Name**: Photo Poet

## Core Features:

- AI Poem Generation: Use generative AI to create poems from uploaded photo's content, using a LLM tool
- Photo Upload: User photo upload via drag and drop
- Poem Display: Display of the generated poem with the uploaded photo
- Poem Copy: Option to copy the poem to clipboard
- Settings: Settings: option to set poem tone/style

## Style Guidelines:

- Primary color: Soft Lavender (#D0B8E7) to evoke creativity and tranquility.
- Background color: Light Gray (#F0F0F5) to provide a gentle, non-distracting backdrop.
- Accent color: Dusty Rose (#D6A2A2) to highlight interactive elements.
- Font pairing: 'Belleza' (sans-serif) for headlines, 'Alegreya' (serif) for body text.
- Use line-art icons to maintain a clean and elegant design.
- Gentle fade-in animations for the poem and photo elements.
